# AZM Book 0 — Assembler Manual

This is the definitive reference for AZM, an enhanced Z80 assembler with modern programming features. It documents the source format, syntax, directives, expressions, layout types, register contracts, op declarations, compatibility features, diagnostics and output formats that define AZM source files.

Use this book when you want the exact assembler rules. If you want a guided introduction to Z80 programming first, start with [AZM Book 1 — Z80 Fundamentals](../book1/index.md).
