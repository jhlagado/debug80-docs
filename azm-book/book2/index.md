# AZM Book 2 — Algorithms and Data Structures

This book applies the Z80 and AZM foundations from Book 1 to larger assembly programs. It works through sorting, searching, strings, bit manipulation, records, recursion, composition, pointer structures and a capstone backtracking problem.

The examples stay close to the machine: every branch, register convention and memory access remains visible in the assembly listing.
