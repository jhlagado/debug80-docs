# Chapter 4 — Ports and Output

This chapter will connect Book 1's `in` and `out` instructions to TEC-1G hardware.

Planned coverage:

- the difference between memory addresses and I/O ports
- why named port constants matter
- first writes to display hardware
- reading output effects in Debug80's TEC-1G panels
- common mistakes: wrong port, wrong bit, stale display state

The chapter will introduce only the ports needed for the first output exercises, then leave the full port map for reference sections.
