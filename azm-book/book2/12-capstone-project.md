# Chapter 12 — Capstone Project

This chapter will finish Book 2 with a complete small TEC-1G program.

Planned requirements:

- use Debug80 as the development loop
- start from a clear MON-3-compatible memory layout
- read keypad input
- draw changing state on the 8x8 RGB matrix
- show status text on the LCD
- show a numeric value on the six-digit display
- play at least one speaker cue
- keep hardware service regular enough that the display remains stable

The capstone should be smaller than Tetro or Pacmo. Its job is to prove that you can combine the hardware features yourself before reading the larger case studies as code tours.
