# Chapter 17 — Beyond the Core

This chapter will map the TEC-1G features that do not fit in the first teaching path.

Planned coverage:

- GLCD as a higher-resolution graphics display
- matrix keyboard mode and how it changes input
- serial workflows
- RTC support
- SD/PATA storage support
- expansion memory and cartridge behavior
- what Debug80 currently emulates
- which features deserve later chapters or a separate follow-up book

The core book should stay focused: MON-3, keypad, LCD, seven-segment display, sound, the 8x8 RGB add-on, and complete programs. This final chapter keeps the larger platform visible without overloading the first pass.
