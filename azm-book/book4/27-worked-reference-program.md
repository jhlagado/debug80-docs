# Chapter 27 — Complete Worked Reference Program

The manual should end with one medium-sized program that uses the core AZM surface in context. The point is not to teach Z80 from scratch, but to show experienced programmers how the syntax fits together.

## Planned sections

- Program goals
- Source file structure
- Constants and enums
- Layout types and storage
- Subroutine contracts
- Op declarations
- Listings and symbols
- Output artifacts
- Portability checks
