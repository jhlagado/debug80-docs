# Preface

This manual is for programmers who already know assembly language and need a compact path into AZM syntax and tooling. It assumes you can read ordinary Z80 source and want to know how AZM spells constants, labels, storage, layout types, register contracts, op declarations, and output formats.

- What it does not teach
- How this manual relates to the three AZM teaching books
