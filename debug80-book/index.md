# Debug80 Books

The Debug80 books teach the Debug80 workflow for Z80 and TEC-1G development in VS Code.

Book 1 takes you from installation through to debugging a TEC-1G program and sending the generated HEX file to hardware. Book 2 is an in-development outline for programming the TEC-1G itself with Debug80 as the working environment.
